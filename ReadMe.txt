Как запустить корпус:

1. Необходимо создать локальный сервер. Есть несколько способов:
    1) а) скачать и установить python
       b) нажать сочетание клавиш Windows + R
       c) ввести в появившемся окне "cmd"
       d) в открывшейся консоли ввести команду: python -m http.server 8000

    2) a) скачать и установить Microsoft Visual Studio
       b) установить расширение Live server
       c) открыть файл index.html с помощью Microsoft Visual Studio
       d) нажать кнопку "Go Live" в нижнем правом углу

2. Зайдите в браузер и введите в адресной строке следующий адрес: http://localhost:8000 ИЛИ открыть файл index.html через браузер


3. Введите ключевое слово для поиска фразеологизма. Доступны сортировка по языкам, семантике, стилистике, происхождению. К большинству фразеологизмов добавлены комментарии с пояснениями.

1.0 -> основной функционал, поиск по фильтрам, ключевым словам

1.1 ->  - фильтр по происхождению (метафора, жаргон, мифологема и тп)
        - фильтр по исконности/заимствованности пословиц
        - поиск по комментарию (для выделения дисфемизмов)
