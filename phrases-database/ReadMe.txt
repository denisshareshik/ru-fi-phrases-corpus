Как запустить корпус:

1. Полностью скачайте архив; 

2. Необходимо создать локальный сервер. Есть несколько способов:
    1) а) скачать и установить python
       b) нажать сочетание клавиш Windows + R
       c) ввести в появившемся окне "cmd"
       d) в открывшейся консоли ввести команду: python -m http.server 8000

    2) a) скачать и установить Microsoft Visual Studio
       b) установить расширение Live server
       c) открыть файл index.html с помощью Microsoft Visual Studio
       d) нажать кнопку "Go Live" в нижнем правом углу

3. Зайдите в браузер и введите в адресной строке следующий адрес: http://localhost:8000 ИЛИ открыть файл index.html через браузер;


4. Введите ключевое слово для поиска фразеологизма. Доступны сортировка по языкам, семантике, стилистике, происхождению. К большинству фразеологизмов добавлены комментарии с пояснениями.
